## What is Convex UI?

Convex UI is a collection of **copy-paste components** built on [shadcn/ui](https://ui.shadcn.com) that integrate seamlessly with [Convex](https://convex.dev). Each component comes with:

- React frontend components (styled with Tailwind CSS)
- Convex backend functions (queries, mutations, schema)
- Full TypeScript support

## How it works

<Command name="password-based-auth-nextjs" />

This single command:

1. **Installs dependencies** - Convex, auth packages, UI components
2. **Creates React components** - Login forms, buttons, layouts
3. **Sets up Convex backend** - Schema, queries, mutations, auth config
4. **Configures environment** - Environment variables template

<Callout type="info" className="mt-6">
  **Your code, your control.** Unlike traditional libraries, these components
  are copied directly into your project. Customize anything - styles, logic,
  backend functions.
</Callout>

## Why Convex UI?

### Real-time by default

Convex provides reactive queries out of the box. When data changes, your UI updates automatically - no polling, no WebSocket setup, no cache invalidation.

<CodeBlock language="tsx">
  {`// This automatically re-renders when messages change
const messages = useQuery(api.messages.list, { roomId: "general" });`}
</CodeBlock>

### Full-stack in one place

Each block includes both frontend and backend code. No separate API layer to maintain.

<CodeBlock language="text">
  {`your-project/
├── src/components/
│   └── login-form.tsx      # React component
└── convex/
    ├── auth.ts             # Auth functions
    └── schema.ts           # Database schema`}
</CodeBlock>

### Framework support

Works with your preferred React framework:

| Framework      | Auth | Storage | Realtime |
| -------------- | ---- | ------- | -------- |
| Next.js        | ✓    | ✓       | ✓        |
| React SPA      | ✓    | ✓       | ✓        |
| React Router   | ✓    | ✓       | ✓        |
| TanStack Start | ✓    | ✓       | ✓        |
